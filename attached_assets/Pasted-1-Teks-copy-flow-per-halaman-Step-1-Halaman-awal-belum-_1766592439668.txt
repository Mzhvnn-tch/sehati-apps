3. Contoh komponen React per portal (struktur kata + code)
Misal kamu punya dua halaman:

Patient landing (pakai “Get started”)
tsx
// src/pages/patient-landing.tsx
import { useLocation } from "wouter";

export default function PatientLanding() {
  const [, navigate] = useLocation();

  return (
    <section>
      <h1>Welcome to SEHATI Patient Portal</h1>
      <p>Access your medical records securely on-chain and manage your health identity.</p>

      <button onClick={() => navigate("/patient")}>
        Get started
      </button>
    </section>
  );
}
Patient page berikutnya – Connect Wallet (AppKit + wagmi)
tsx
// src/pages/patient-dashboard.tsx
import { AppKitButton } from "@reown/appkit/react";

export default function PatientDashboard() {
  return (
    <section>
      <h1>Connect your wallet to continue</h1>
      <p>
        SEHATI uses your wallet as a secure on-chain identity to access your medical profile.
      </p>

      {/* Tombol connect wallet dari AppKit */}
      <AppKitButton />
    </section>
  );
}
Kalau kamu mau teks tombolnya tetap “Get started” tapi fungsinya connect wallet, kamu bisa pakai hook useAppKit dan atur copy sendiri:

tsx
import { useAppKit } from "@reown/appkit/react";

export default function PatientDashboard() {
  const { open } = useAppKit();

  return (
    <section>
      <h1>Welcome back, patient</h1>
      <p>
        To continue to your dashboard, please connect your wallet as your SEHATI identity.
      </p>

      <button onClick={() => open()}>
        Get started by connecting your wallet
      </button>
    </section>
  );
}
Kata‑katanya jelas:

“Get started by connecting your wallet” → user paham bahwa “start” di step ini = connect wallet.
